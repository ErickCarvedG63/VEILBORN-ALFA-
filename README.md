# VEILBORN-ALFA-
VEILBORN — Prototype 0.1
Un juego de plataformas inspirado en Hollow Knight, desarrollado en Java utilizando Swing. Incluye movimiento fluido, combate básico, un jefe, música, parallax y sistema de pausa. Esta es una versión prototipo/alpha centrada en mecánicas base y pruebas de gameplay.

Características
Movimiento: caminar, saltar, dash y ataque.

Enemigos básicos y un jefe con IA simple.

Mundo con plataformas, parkour y zonas peligrosas.

Menú inicial con fade-in.

Final al derrotar al jefe.

Música por fases con sistema corregido de audio.

Controles
A / D → Mover

W → Saltar

J → Atacar

Shift → Dash

Enter → Iniciar juego

ESC → Pausar / Reanudar / Salir en menú o final

Estado del proyecto
Versión: Prototype 0.1 (Early Alpha)
Enfocado en jugabilidad básica; aún faltan animaciones, historia y niveles completos.
